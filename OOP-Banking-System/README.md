# Setup
`git clone git@github.com:ibury08/Springboard_DE.git`  
`cd Springboard_DE/OOP-Banking-System`  
`make init && make test`  

# Starting the app
`make run`  
Or in debug mode for full stacktrace and dev database using `make debug`
